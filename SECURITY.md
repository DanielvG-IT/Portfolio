# Security Policy

This repository contains the source for my personal portfolio website,
[danielvanginneken.nl](https://danielvanginneken.nl). It is a single,
continuously deployed site, so only the latest version is maintained.

## Supported Versions

| Version                 | Supported          |
| ----------------------- | ------------------ |
| `main` (deployed)       | :white_check_mark: |
| Any older commit / tag  | :x:                |

## Reporting a Vulnerability

If you discover a security issue, please report it **privately** — do not open a
public issue or pull request.

- **Preferred:** [GitHub private vulnerability reporting](https://github.com/DanielvG-IT/Portfolio/security/advisories/new)
- **Email:** hello@danielvanginneken.nl

Please include, where possible:

- A description of the issue and its potential impact
- Steps to reproduce (affected URL, page, or file)
- Any relevant logs, screenshots, or proof of concept

### What to expect

- I aim to acknowledge your report within **3 business days**.
- If the issue is confirmed, I'll work on a fix and keep you updated on progress.
- Once resolved, I'm happy to publicly credit you for the disclosure — let me
  know if you'd prefer to stay anonymous.
- If the report is declined, I'll explain why.

Thank you for helping keep the site and its visitors safe.
